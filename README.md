## Kurohana

# Overview
Kurohana is a production-grade, multipurpose Discord bot built with the modern and performant NetCord library in C#. Designed for aesthetic quality and powerful functionality, it aims to provide Discord communities with advanced server customization, robust moderation, secure management, and an engaging economy system.

# Setup & Installation
Requirements
- .NET 8.0+ SDK
- A Discord Bot Token from the Discord Developer Portal

# Getting Started
Clone the repository:

```bash
git clone https://github.com/yourusername/your-bot-name.git
cd your-bot-name
```

# Add your configuration:
Create an appsettings.json file (excluded from the repository for security) and insert your bot's configuration:

```json
{
  "Token": "your-bot-token",
  "DefaultGuildId": "your-server-id"
}
```
# Build and run the bot:

```bash
dotnet build
dotnet run
```
